# Hospital Management System

Welcome to the Hospital Management System project! This Spring Boot application is designed to manage doctors, patients, and nurses within a hospital environment. The project utilizes a HashMap-based in-memory database to store and retrieve data.

## Project Overview

The Hospital Management System aims to simplify the management of hospital-related entities such as doctors, patients, and nurses. The application provides RESTful APIs to perform CRUD operations on these entities, making it easy to manage and retrieve information.

## Technologies Used

- Java
- Spring Boot
- Spring Web
- Lombok
- In-memory HashMap-based database

## Features

- CRUD operations for doctors, patients, and nurses.
- RESTful APIs for managing hospital entities.
- Utilizes the power of Spring Boot for streamlined development.

## Getting Started

To run this project locally, follow these steps:

1. Clone this repository to your local machine.
2. Open the project in your preferred Java IDE.
3. Run the `HospitalManagementSystemApplication` class to start the Spring Boot application.
4. Access the APIs using a tool like Postman.

## Usage

- Use the provided RESTful APIs to interact with doctors, patients, and nurses.
- Create, Read, Update, and Delete entities through the respective endpoints.
- Manage the hospital-related data seamlessly using the application.
